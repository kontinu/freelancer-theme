docker


Docker es la plataforma para aplicaciones del futuro, con contenedores Docker podras obtener algunas de estas caracterísiticas a tus equipos de desarrollo y operaciones:

- Agilidad

- Portabilidad

- Seguridad

- Reducir costos de infrastructura

- Simplicidad en Infrastructura


También es la plataforma que te ayudara a migrar a Microservicios , Mejorar tu proceso actual de Integración Continua y Entrega Continua, 3 de los aspectos más importantes hacia Devops.
